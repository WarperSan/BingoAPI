BingoAPI is a .NET library to communicate with BingoSync's servers directly through code.

## Features

- Creating rooms
- Joining rooms
- Obtaining the board
- Sending / Receiving messages
- Marking / Clearing goals
- Changing team
- Revealing the card
- Obtaining the room feed
